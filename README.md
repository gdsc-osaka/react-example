# react-example

GDG on Campus University of Osaka - チーム開発はじめの一歩 ハンズオン用リポジトリ。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開きます。

## ハンズオンの流れ

1. このリポジトリを clone する
2. 自分のブランチを切る (`git switch -c feature/add-<your-name>`)
3. `src/App.jsx` の `participants` 配列に**自分の名前を追加**する
4. commit して push する
5. GitHub で Pull Request を作成する

```diff
  const participants = [
    'Alice',
+   'YOUR_NAME',
  ]
```
